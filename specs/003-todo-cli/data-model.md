3→## Todo Entity
4→
5→**Description**: Represents a single task.
6→
7→**Fields**:
8→
9→- `id`: A unique integer identifier for the todo (auto-increment).
10→- `title`: A string describing the task (required, e.g., "Buy groceries").
11→- `completed`: A boolean indicating whether the task is finished (default: `false`).
